CleanYourMac 是 CleanMyMac 的开源替代品，作为一个真正的原生 SwiftUI 应用重建——不是 Electron 套壳，也不是 shell 脚本。它可回收磁盘空间、清除隐私痕迹、彻底卸载 App 及其残留，并检查您的安全配置，且绝不会在您未审阅的情况下删除任何内容。采用 MIT 许可证，永久免费：无订阅、无账户、无推销。

> *与 MacPaw 无任何关联。“CleanMyMac 开源替代品”仅作为名义参照。*

## 为什么选择 CleanYourMac

- **开源且免费 — 无订阅费。** 基于 MIT 许可证且完全免费，没有账户，也没有烦人的推销。阅读代码、Fork 它、信任它——无需为无法审查的清理软件支付每年约 40 美元的订阅费。
- **真正的原生。** 真正的 SwiftUI 构建，带有毛玻璃侧边栏和克制且具设计感的「曜石」美学——一款感觉就像生来属于 Mac 的应用。
- **诚实的设计，无暗黑模式（套路）。** 没有任何东西会在未经您审阅的情况下被删除，所有数值都是真实的，并且破坏性操作是默认关闭且需要手动开启的。每个隐私清理项都带有一条诚实的影响提示，例如“会让您退出登录”或“清除您的命令历史”。
- **架构保证安全。** 声明式可清理白名单、受保护路径黑名单、符号链接归一化和最小路径深度守卫，这意味着它在物理层面上无法将您置于危险之中，不会误删主目录、文档或「/」。所有删除都移至废纸篓，可撤销。

## 功能特点

**1. 智能扫描 (Smart Scan)** — 一键并发运行所有模块，然后向您展示结果以供审阅。未经您的批准，不会删除任何内容，并且总量是真实的。

**2. 系统垃圾 (System Junk)** — 从用户缓存（`~/Library/Caches`）、应用程序日志、Xcode DerivedData 和开发工具缓存（npm / Gradle / CocoaPods）中回收磁盘空间。它清理的所有内容都是用户所有的，不需要 sudo 或 root 权限，并且系统会自动安全地重新生成它们。

**3. 卸载器 (Uninstaller)** — 删除 App 及其所有残留：Application Support、缓存、偏好设置（含 ByHost）、容器、群组容器、保存的状态、日志、Web 存储、WebKit 数据、Cookie、启动代理和应用脚本。每个匹配项都带有置信度级别——**高 (HIGH)**（精确的 Bundle 标识符）或**中 (MEDIUM)**（通过应用名称匹配，建议删除前先核对），并且它拒绝触碰 Apple/系统应用。

**4. 大文件与旧文件 (Large & Old Files)** — 扫描大文件和旧文件，以便您可以审阅并清理不再需要的内容。

**5. 隐私 (Privacy)** — 二合一模块：
- **痕迹清除**：涵盖 Safari、Chrome、Edge、Brave、Vivaldi、Firefox、Arc、Opera、Opera GX、Chromium 及自动检测 of Electron 应用——外加 macOS 子系统，如最近使用项、QuickLook 缩略图缓存、窗口保存状态、下载“来源”隔离数据库、Shell/终端历史记录，以及崩溃与诊断报告。每个项目都有诚实的影响提示，破坏性项目（Cookie、站点数据、打开的会话、窗口状态、Shell 历史）默认关闭。
- **安全与隐私审计**：标记真实的配置风险：启用了远程登录 (SSH)、屏幕共享开启、文件共享开启、防火墙关闭、FileVault 关闭、共享 Mac 分析数据开启、启用了访客用户、AirDrop 对所有人可见、全局可读的 SSH 私钥 / AWS 凭证 / `~/.netrc`，以及可能残留在 Shell 历史中的密钥。

**6. 邮件附件 (Mail Attachments)** — 回收 Apple Mail 为您打开或保存过的附件在本地留下的副本。原件仍保存在邮件里，因此这些副本可以安全移除——并且它们同样进入废纸篓，随时可恢复。

**7. 废纸篓 (Trash Bins)** — 按大小排序检视废纸篓里的内容，让您只清空占空间的大块头。这是唯一一个删除真正不可逆的模块（这些项目本来就已被删除过一次），确认前界面会用最直白的话讲清楚。

**8. 优化 (Optimization)** — 只读检视开机启动项：用户启动代理、系统级第三方代理和守护进程——带有"登录时运行/已停用"标记、在访达中显示，以及跳转到系统设置登录项的快捷入口。它绝不改动任何东西。

**9. 维护 (Maintenance)** — 运行一组固定的、免 sudo 的安全刷新任务：刷新 DNS 缓存、重建 Launch Services 数据库（修复"打开方式"菜单里的重复项）、重置 Quick Look 缩略图缓存、重启访达或程序坞。固定命令、固定参数——绝不经过 shell，绝不掺入您的输入。

**10. 空间透镜 (Space Lens)** — 只读的文件夹空间浏览器：扫描主目录，按大小排列每个文件和文件夹，沿面包屑路径逐层下钻。它只看，不删。

## 安全保障

在这里，安全不是一个承诺，而是产品架构。四道防线协同工作，使该应用在**物理层面上无法**删错东西：

- **声明式可清理白名单**。可清理的路径是一个固定的、已知安全的白名单——绝不是从要删除的项目中动态推导出来的。如果一个路径不在白名单上，它就是绝对的禁区。
- **受保护路径黑名单**。某些路径**绝不会**被触碰：`/`、`/System`、`/Library`、`/usr`、`/Applications` 以及您的整个主目录——包括文档、桌面、下载、电影、音乐、图片、邮件、信息、钥匙串、照片和 Application Support。
- **符号链接归一化**。在执行任何操作之前，每个路径都会被完全解析（canonicalization），因此没有任何东西可以通过符号链接“逃逸”出允许的根目录并到达受保护的目录。
- **最小路径深度守卫**。一个路径必须足够深才能成为真实的特定目标。该应用永远不会瞄准像 `/` 或 `/Users/me` 这样浅层且灾难性的目录。

此外：所有清理项都移至**废纸篓**（可撤销，而非永久删除）——唯一的诚实例外是"废纸篓"模块，它的职责本身就是清空废纸篓，并会在您确认前明确说明"永久删除、不可恢复"。该应用绝不移除密码、自动填充或保存的卡片、书签、个人文档、照片或钥匙串数据。此模型非常诚实，并经过了对抗性审查——因为诚实本身就是我们的品牌。

**它不做的事情**：没有病毒扫描或恶意软件删除（应用中没有防病毒功能），也没有“为 Mac 加速/释放内存”的玄学魔法。它的价值是真实且清晰的：安全释放磁盘空间、清理隐私痕迹、干净地卸载 App，以及体检安全设置。
